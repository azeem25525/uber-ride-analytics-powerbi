# Uber Ride Analytics Dashboard – Power BI

## Objective
Analyze Uber ride bookings to track booking value, success bookings, and distance travelled across vehicle types.

## Tools
Power BI, DAX, Excel

## KPIs
- Total Booking Value
- Success Booking Value
- Avg. Distance Travelled
- Total Distance Travelled
- Vehicle Type Performance
